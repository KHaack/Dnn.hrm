# Dnn.hrm
DotNetNuke Modules for Human Resource Management.

## Prerequisites
### .NET Framework
-  4.7.2+

### Web Server
-  Microsoft IIS 7.5+

### Database Server
-  Microsoft SQL Server 2016+

### DotNetNuke
-  DNN 9.6.2+

### Browser Support
-  Chrome
-  Firefox
-  IE 11
-  Microsoft Edge
-  Safari for Mac

### Recommented IDE
-  Visual Studio 2019+
